☁️ Cloud Notes App

📌 Description
Developed a cloud-based notes application that allows users to create, store, and manage notes anytime with secure storage and easy accessibility.

🚀 Features
- Create, edit, and delete notes
- Cloud storage using Firebase
- Real-time updates
- Simple and user-friendly UI

🛠️ Tech Stack
- HTML, CSS, JavaScript
- Firebase

▶️ How to Run
1. Clone the repo
2. Open project folder
3. Run using browser or deploy via Firebase
